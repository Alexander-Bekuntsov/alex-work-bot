# Alex Telegram Form Bot

## Описание
Telegram-бот для проведения опросов и генерации отчетов по результатам

---

## Содержание архива
Архив содержит весь исходный код проекта, включая:
- Dockerfile (с multi-stage сборкой)
- docker-compose.yml
- Файлы с исходным кодом Java приложения
- README.md (этот файл)
- Файл локализации бота
- Также все другие необходимые файлы для запуска

---

## Инструкция по сборке и запуску

### 1. Распаковка проекта
Распакуйте архив в удобную папку на вашем компьютере

### 2. Настройка переменных окружения
Создайте файл `.env` рядом с `docker-compose.yml` и заполните его следующим образом:
- `BOT_TOKEN` - токен вашего Telegram-бота (полученный у BotFather)
- `BOT_USERNAME` - username вашего Telegram-бота (например, `work_name_bot` если @work_name_bot)
- `DB_NAME` - название базы данных (например, `telegram_bot_db`)
- `DB_USER` - пользователь базы данных PostgreSQL (например, `alexalexuser`)
- `DB_PASSWORD` - пароль пользователя базы данных
- `DB_URL` - `jdbc:postgresql://db:5432/${DB_NAME}` (вместо `${DB_NAME}` вставьте DB_NAME)

### 3. Сборка и запуск с помощью Docker Compose

#### 1. Откройте терминал в папке с проектом и выполните команду `docker-compose up --build`
#### 2. Эта команда соберет образ приложения и запустит два сервиса:
- bot - Telegram-бот
- db - PostgreSQL с персистентным хранилищем
#### 3. После полного запуска остановите приложение с помощью `Ctrl+C` или другой команды и затем выполните команду
- `docker-compose up`

После успешного запуска бот будет полностью готов к работе

### 4. Проверка работы
- В Telegram найдите вашего бота и начните с ним диалог. Используйте команду /start
- Используйте команды бота

---

## Описание команд бота
- `/start` — запуск бота и приветственное сообщение
- `/form` — начать заполнение формы (например, опрос или регистрация)
- `/report` — сгенерировать и получить отчет
